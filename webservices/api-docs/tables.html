API Count Tables Webservice

Request

URL


        GET {url}/api-tables.php?output=xml&table=3                 /* get table of 3 from 1 to 10 in XML format */
        GET {url}/api-tables.php?output=json&table=7                /* get table of 7 from 1 to 10 in JSON format */
        GET {url}/api-tables.php?output=xml&table=5&start=3&end=17  /* get table of 5 from 3 to 17 */

Params

Param Name Description
table The count table
Optional, default is 1
start Start counting
Optional, default is 1
end End counting
Optional, default is 10
output Response in XML or JSON format
Optional, default is JSON

Response

XML format


      <?xml version="1.0"?>
      <results>
          <result>
              <number>1</number>
              <table>3</table>
              <outcome>3</outcome>
          </result>
          <result>
              <number>2</number>
              <table>3</table>
              <outcome>6</outcome>
          </result>
      </results>

JSON format


      {
          "results": [
              {
                  "number": 1,
                  "table": 3,
                  "outcome": 3 
              },
              {
                  "number": 2,
                  "table": 3,
                  "outcome": 6
              }
          ]
      }
