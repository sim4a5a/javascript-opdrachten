API Employees Webservice

Request

URL


        GET {url}/api-employees.php?output=xml             /* get one random employee in XML format */
        GET {url}/api-employees.php?output=json&amount=5           /* get one random employee in JSON format */
        GET {url}/api-employees.php?output=xml    /* get 5 random employees */

Params

Param Name Description
amount The amount of random employees
Optional, default amount is 1 employee
output Response in XML or JSON format
Optional, default is JSON

Response

XML format


      <?xml version="1.0"?>
      <employees>
          <employee>
              <name>Allistair M. Hays</name>
              <company>Nec Ante Blandit Foundation</company>
          </employee>
          <employee>
              <name>Alika Donaldson</name>
              <company>Egestas Company</company>
          </employee>
      </employees>

JSON format


      {
          "employees": [
              {
                  "name": "Allistair M. Hays",
                  "company": "Nec Ante Blandit Foundation"
              },
              {
                  "name": "Alika Donaldson",
                  "company": "Egestas Company"
              }
          ]
      }
