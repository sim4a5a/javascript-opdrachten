API Countries Webservice

Request

URL


        GET {url}/api-countries.php?output=xml             /* get all countries in XML format */
        GET {url}/api-countries.php?output=json            /* get all countries in JSON format */
        GET {url}/api-countries.php?output=xml&id=3        /* get country with id 3 */

Params

Param Name Description
id The country ID.
Optional, when no ID is given all countries will be shown
output Response in XML or JSON format
Optional, default is JSON

Response

XML format


      <?xml version="1.0"?>
      <countries>
          <country>
              <id>6</id>
              <name>Indonesia</name>
          </country>
          <country>
              <id>33</id>
              <name>Mexico</name>
          </country>
      </countries>

JSON format


      {
          "countries": [
              {
                  "id": 6,
                  "name": "Indonesia"
              },
              {
                  "id": 33,
                  "name": "Mexico"
              }
          ]
      }
